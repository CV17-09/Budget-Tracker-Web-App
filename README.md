## 📖 Overview

The **Budget Tracker Web App** was built to simplify personal finance management by giving users an easy way to monitor their income, expenses, and overall financial health.

Instead of relying on spreadsheets or handwritten notes, users can manage transactions, track their current balance, and gain valuable insights through interactive charts and financial summaries.

This project demonstrates practical front-end development skills including:

- Dynamic JavaScript functionality
- Responsive web design
- Interactive UI components
- Data visualization
- Financial dashboard development

---

## ✨ Features

### 💵 Financial Management

- Add income transactions
- Add expense transactions

- Real-time account updates

- Spending summaries
